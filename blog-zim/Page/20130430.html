Write Blog Using Zim & Github Pages

Created Tuesday 30 April 2013
@github_pages @blog @web

Gen static pages with JekyII may be the offical company for github pages. but you must learn a new syntax for a markup language. For making things easier, I choose zim to gen the static pages.

How to combine zim and github pages?

  1. create a fresh repo named yourname.github.io on github.com
  2. create github branch: master
  3. git pull from your github
  4. generate static htmls with zim 'export' in master branch.
  5. commit all and push all.
  6. done

note:
project pages use gh-pages branch.
personal pages use master branch.

so others can access my new blog on picospuch.github.com now. haha.
